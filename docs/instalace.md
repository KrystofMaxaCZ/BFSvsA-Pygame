# Instalace a spuštění
Níže naleznete postup pro správnou instalaci a spuštění aplikace.

## Požadavky
Před spuštěním projektu je nutné mít nainstalováno:

- Python 3.10 nebo novější

## Postup spuštění
1. Vytvoření virtuálního prostředí (venv):
   ```bash
   python -m venv venv
   ```
2. Aktivace virtuálního prostředí:
   ```bash
   venv\Scripts\activate
   ```
3. Instalace závislostí:
   ```bash
   pip install -r requirements.txt
   ```
4. Spuštění projektu:
   ```bash
   python main.py
   ```
